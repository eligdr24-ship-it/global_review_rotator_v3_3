Global Review Rotator v3.4

Added:
- Full calendar date search on Admin and Operator Analytics dashboards.
- Default dashboard range is current week, Sunday to Saturday.
- Quick filters: This Week, Today, Last 7 Days, Last 30 Days, Last 90 Days, Last 1 Year.
- Dashboard counters, charts, history, operator leaderboard, business-tab analytics and Excel export follow the selected date range.
- Back buttons added to Admin, Worker, and Analytics pages.
- Analytics opened from Admin can return back to Admin; normal analytics returns back to the operator work dashboard.

Kept:
- Worker dashboards remain open without password.
- Analytics pages require operator password.
- Admin dashboard requires admin password.
- History is preserved after Excel uploads.
- Multi-sheet fair distribution by link tab/business tab.
